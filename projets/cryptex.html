Projet Web — SAE 203

Cryptex

Dashboard de suivi de cryptomonnaies en temps réel

Contexte SAE 203 — BUT MMI
Établissement IUT de Troyes
Type Application Web
Année 2025
Description

Cryptex est une application web permettant de suivre les cours des cryptomonnaies en temps réel. Le projet repose sur la consommation d'une API publique pour récupérer dynamiquement les données de marché et les afficher dans une interface claire et réactive.

L'accent a été mis sur la lisibilité des données, la mise à jour en temps réel et une expérience utilisateur fluide sans rechargement de page.

Technologies utilisées
HTML5 CSS3 JavaScript REST API Fetch API JSON
